# Paginas-web-

Sitio web estático de una sola página para un colegio (demo de ventas).
Hecho con **HTML, CSS y JavaScript vanilla**: sin frameworks, sin build, sin backend.

## Archivos

| Archivo | Para qué sirve |
|---|---|
| `index.html` | Estructura y textos del sitio (aquí editas casi todo) |
| `styles.css` | Diseño y colores (edita la sección `:root` para cambiar la paleta) |
| `script.js` | Menú móvil y año automático del footer |

## Cómo verlo

Haz **doble clic** en `index.html` y se abre en tu navegador. No necesita instalar nada.

## Qué reemplazar antes de vender

Abre `index.html` y busca la palabra **`REEMPLAZAR`**: cada comentario te dice qué cambiar.
Lo principal:

- Nombre del colegio y lema
- Logo y fotos (cajas grises "LOGO AQUÍ" / "FOTO AQUÍ")
- Textos de Nosotros, niveles e inscripciones
- Dirección, teléfono (hoy dice `12345678`), correo y WhatsApp
- Mapa de Google Maps (iframe) y botón "Ver en Google Maps"
- Links de Instagram y Facebook

## Cómo publicarlo gratis

- **Netlify:** arrastra la carpeta a https://app.netlify.com/drop
- **GitHub Pages:** Settings → Pages → Branch `main` → carpeta `/ (root)`
